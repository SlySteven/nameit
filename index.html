Name It

You'll have three minutes to name something in each of ten categories. Answers must start with a particular letter. For example, if the letter is C for "Items In A Refrigerator," you could answer "Cucumber," "Cool Whip," or "Cold Cut."

Repeating letters gives an extra point per instance. No repeating parts of answers. Players can veto answers with a vote. There are a few more advanced rules as well.

Adjectives must change the core identity of the term, e.g. "Chicken Sandwich" or "Creamed Corn." Superfluous adjectives are not allowed, e.g.Â "Cold Water" or "Cubed Pineapple."

The answer must be commonly associated with the category. For "Things That Jump," you can't put a celebrity (because they have the capability of jumping) unless they're known for jumping (like a basketball player).

For peoples' names, the first or last name can be used to match the letter.

Gather up your players, then hit SYNC at the same time.

When everyone has the same Game ID (), hit START at the same time!
